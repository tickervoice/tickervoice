
  # ChangeAlias
  
  Note: alias support,sales,sysadmin,admin,administrator  - these are server reserved alias 
  
  - set one alias
  ~~~bash 
  alias alias1
  ~~~

  - set multiple alias, first one will be used by default if not specified in use
  ~~~bash 
  alias alias1,alias2
  ~~~
